What is Object-Oriented Programming (OOP)?


It simply means using object in programming.

The OOP used to be so scary to me. When I started to program using C# in 2007, I always try to avoid the OOP in all ways.

But as time goes by, I realize that it is a "must" skill if I really want to be a good programmer who can develop the code/ software that is easy to reuse and maintain.

I am still practicing the OOP design in my own code, and I find that OOP design is not intimidating anymore for me. I would like to share my favorite online learning resources that I used self-taught myself.
